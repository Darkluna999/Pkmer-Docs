# Obsidian 样式：书签列表样式

## 引言

在 AnuPpuccin 主题的 GitHub Discussion 中看到其他大佬分享的列表 CSS 样式，觉得还蛮漂亮，就自己扩充了一下，完善了一点效果。

> [!warning]+ 注意
> 由于个人代码能力有限，本 CSS 效果仅适配于源码和预览模式。欢迎强力大佬 PR。
