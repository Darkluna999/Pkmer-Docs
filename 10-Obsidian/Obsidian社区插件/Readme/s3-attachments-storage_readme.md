## 开始使用

- 克隆这个仓库。
- 运行 `npm i` 安装依赖。
- 运行 `npm run build` 编译成 `main.js`。

## 手动安装插件

- 将 `main.js` 和 `manifest.json` 复制到你的 vault 中的 `VaultFolder/.obsidian/plugins/your-plugin-id/` 目录下。

## 功能列表

支持的文件类型（受 Obsidian 允许链接的文件类型限制）：

- 图像文件（.ico, .png, .jpg, .gif）。
- 视频文件（.mp4）。
- 音频文件（.mp3, .wav）。
