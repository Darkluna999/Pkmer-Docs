# 连字

在 Obsidian 中启用文本对齐和连字。

使用 `navigator.language` 来确定连字的主要语言。

可以通过在文件的前置元数据中添加 `lang` 键来更改每个文件的语言：

```md
---
lang: de
---
```

该插件的 CSS 将规则应用于预览模式中的 `p`、`ol` 和 `ul` 元素，

但对于外部链接，我认为在连字时看起来很丑。
