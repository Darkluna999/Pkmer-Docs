## Obsidian 富链接插件

这是一个为 Obsidian 设计的富链接插件，它允许你将笔记中的 URL 转换为像 Notion 一样的富链接预览。

### 如何使用

* 输入一个链接
* 选择链接
* 打开命令面板
* 选择**创建丰富链接**
