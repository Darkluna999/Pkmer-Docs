## 安装方法

1. 在 [Obsidian](https://www.obsidian.md) 的设置中找到 **Community Plugins**，然后**禁用**安全模式
2. 点击 **浏览** 并搜索 "Open with"
3. 点击安装
4. 在 **Community Plugins** 标签中切换插件开启状态
