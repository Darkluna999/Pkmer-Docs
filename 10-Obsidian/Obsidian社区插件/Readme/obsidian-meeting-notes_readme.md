# Obsidian 会议笔记插件

在为每次会议创建相同的结构、复制粘贴内容感到沮丧之后，我决定创建这个插件。

## 设置

默认情况下，插件使用名称“MeetingNotes”作为“会议文件夹位置”。您可以在设置中将其更改为任何您喜欢的名称。

## 演示

在指定的“会议文件夹位置”创建文件时，插件会自动填充模板，以便生成一致的会议记录。
