# Obsidian Metal Archives 插件

这是一个为 Obsidian 设计的插件，允许你通过从 Metal Archives（<https://www.metal-archives.com/）获取数据来创建关于金属乐队和专辑的笔记。>

该插件提供了两个命令：

- 搜索乐队并创建笔记
- 搜索专辑并创建笔记
