# obsidian-toggle-meta-yaml-plugin

**提示：Obsidian 的最新版本已经提供了元数据 yaml 折叠功能。**

这是一个简单的插件，用于切换元数据 yaml。

该插件只实现了一个功能，即在你的 markdown 中显示或隐藏元数据 yaml。

为了方便切换，你可以将命令 `toggle-meta-yaml: toggle` 绑定到你的热键，比如 `cmd+shift+m`。
