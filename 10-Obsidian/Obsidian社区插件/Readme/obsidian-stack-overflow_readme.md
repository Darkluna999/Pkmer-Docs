# 安装

在 Obsidian 社区插件部分搜索 `Stack Overflow Answers`

## 手动安装

将此文件夹克隆到您的 Obsidian `.plugins` 文件夹中，然后：

1）运行 `npm i` 或 `yarn` 以安装依赖项。

2）运行 `npm run dev` 以启动监视模式下的编译。

# 用法

插入 Stack Overflow 答案的默认快捷键是 `cmd+shift+v`。可以在 `community plugins` 部分进行更改。
