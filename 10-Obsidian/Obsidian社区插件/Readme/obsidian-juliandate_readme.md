## Obsidian 儒略日期插件

这是一个简单的插件，为插入儒略日期添加了一个热键。儒略日期是自儒略纪元开始以来的连续天数计数，主要用于天文学家注释天文事件。

默认热键是 `Ctrl+Shift+J`。

如何编译插件

首先，使用以下命令安装依赖项：

```bash
npm i
```

然后，可以使用以下命令编译插件：

```bash
npm run build
```

这将在项目根目录中创建一个 `main.js` 文件。这是 Julian Date 插件的入口点。
