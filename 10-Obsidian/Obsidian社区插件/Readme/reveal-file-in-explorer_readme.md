# 在资源管理器中显示文件

在资源管理器中显示活动文件的方法：

* 单击标题栏
* 打开文件

折叠不包含活动文件的其他文件夹

设置：

![演示](settings.jpg)

注意：如果禁用“打开文件时显示”，与其相关的其他设置将被隐藏

编辑：添加了一个选项，可以排除某些文件夹在打开时显示
