## 故障排除

Obsidian Vale 目前不支持 Obsidian 13 中引入的新编辑器。

要继续在版本 13 中使用该插件，您可以启用传统编辑器。

1. 打开**设置** -> **编辑器**。
2. 启用**传统编辑器**。
3. 重新启动 Obsidian。
