### 特点

- 将笔记嵌入到幻灯片中
- 在编辑幻灯片时进行实时预览
- 主题允许您更改幻灯片的外观
- 注释允许您更改幻灯片的样式
- 将幻灯片导出为 PDF 文档或 HTML 演示文稿
- **完全支持 Obsidian Markdown 语法**
