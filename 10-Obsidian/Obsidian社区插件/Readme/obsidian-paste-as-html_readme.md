# Obsidian 粘贴为 HTML 插件

1. 安装此插件
2. 在浏览器中复制 HTML
3. 使用快捷键 `ctrl + p` 并执行命令 `paste as html`
4. 网页文本将以 HTML 格式粘贴，并保留原始的 CSS 样式
