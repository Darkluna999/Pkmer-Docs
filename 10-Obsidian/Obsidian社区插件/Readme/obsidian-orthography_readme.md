## 特点

* 在文本中搜索拼写错误
* 显示纠正单词错误的选项
* 一键纠正单词错误

## 对于开发者

### 如何开始开发

1. 在 Obsidian 中创建一个新的保险库（文件夹）。例如 `Orthography`
2. 在终端中，进入 `./Orthography/.obsidian` 文件夹，并在其中创建 `plugins` 文件夹
3. 将存储库克隆到 `plugins` 文件夹中：`git clone https://github.com/denisoed/obsidian-orthography.git`
4. 安装依赖项：`npm i`
5. 运行开发命令：`npm run dev`
