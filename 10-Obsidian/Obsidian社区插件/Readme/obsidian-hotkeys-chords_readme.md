# Obsidian 的快捷键和和弦

该插件允许您设置快捷键和弦（一系列快捷键）来激活 Obsidian 命令。

这使得可以将类似 Emacs 的按键绑定序列（如 `C-x C-s`）绑定到插件配置窗格中的任何可用的 Obsidian 命令（包括其他插件导出的命令）。

如果有任何不清楚或不起作用的地方，请提出问题，但我已经知道我没有太多时间来专门处理它。
