### 如何使用

- 克隆这个仓库。
- 运行 `npm i` 或 `yarn` 安装依赖。
- 运行 `npm run dev` 以启动编译并进入监听模式。

### 如何安装插件

- 将 `main.js`、`styles.css` 和 `manifest.json` 复制到你的 vault `vault/.obsidian/plugins/plugin-id/` 目录下。
