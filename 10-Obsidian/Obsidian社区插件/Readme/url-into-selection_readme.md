## Readme(翻译）

下面是 [[url-into-selection]] 插件的自述翻译

将 URL 粘贴到选定的文本中，使用常规的 `Ctrl/Cmd + V` 键插入链接（URL），以 "notion-style" 的方式。

同样也可以反过来，通过命令面板/快捷键（需要手动设置）将文本插入到选定的链接（URL）中。

# 安装

要安装此插件，请转到“设置 > 第三方插件 > 将 URL 粘贴到选择中”。
