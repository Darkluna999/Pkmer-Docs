# Obsidian 元绑定插件

该插件可以在您的笔记中创建输入字段，并将它们绑定到元数据字段，同步其内容。

允许您在笔记中创建复选框等内容，当选中时，更改元数据字段（例如 `done`）。

或者在电影笔记中创建一个滑块，与名为 `rating` 的元数据字段同步。
