# Obsidian 备忘录插件

一个用于 Obsidian 的插件，允许您同步您的**[Memos](https://usememos.com/)**。

如何使用

只需在插件设置中输入您的 OpenAPI，然后您就可以点击左侧的按钮来同步您的备忘录。

**您可以在备忘录设置中找到您的 OpenAPI。**
