## 概述

Make.md 为你带来了超强的 Obsidian 体验增强功能。自定义顺序对你的文件进行排序，并使用 Spaces 添加文件图标。用 Flow 编辑器编辑内嵌文件。并使用 Maker Mode 为你的文本样式和添加新的 markdown 块而不需要写 markdown。
