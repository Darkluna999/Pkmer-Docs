## 每日笔记查看器

这是一个 Obsidian 插件（<https://obsidian.md）。>

它可以帮助您在一个页面上查看一些最近的每日笔记。

### 演示

![演示](https://cdn.pkmer.cn/covers/obsidian-daily-notes-viewer_2_0.png!pkmer)

### 注意事项

- 请确保已启用核心插件中的每日笔记插件。
- 启用插件时，查看器文件将自动创建。
- 创建每日笔记或更改设置时，查看器文件将自动更新。

### 如何使用

- 启用插件。
- 单击功能区图标跳转到查看器文件。
