# 用法

在代码块中放置您想要的动作类型（例如 `pf2:2` 表示双动作），它将在视图和实时预览中被相应的图标替换。

| 代码    | 动作          |
| ------- | ------------- |
| `pf2:1` | 动作          |
| `pf2:2` | 两个动作      |
| `pf2:3` | 三个动作      |
| `pf2:0` | 自由动作      |
| `pf2:r` | 反应动作      |

## 幻想统计数据块

如果属性被配置为“渲染为 Markdown”，您可以在文本中添加带有动作的代码块，它将被替换为图标。
