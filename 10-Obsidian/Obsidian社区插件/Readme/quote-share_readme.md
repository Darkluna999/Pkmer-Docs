# Obsidian Share

这个插件允许你在 Obsidian 中嵌入任何网站。

- 为您的文本生成渐变图像

## 使用方法

1. 将文本拖放到渐变区域
2. 点击“复制到剪贴板”
3. 将其粘贴到您的笔记或 Twitter 中
