下面是 [[mousewheel-image-zoom]] 插件的自述翻译

Obsidian 鼠标滚轮图像缩放

该插件使您能够通过按住可配置的键（默认为左 Alt 键），在预览模式下悬停在图像上并滚动来增加/减小图像的大小。
