## Readme(翻译）

下面是 [[weekly-review]] 插件的自述翻译

# 每周回顾

我的每周回顾流程包括回顾我本周创建的所有文件，但我发现在 Obsidian 中打开所有这些文件有点繁琐。这个插件可以打开你在过去一周内创建的所有文件（你可以配置要打开多少天的文件）。
