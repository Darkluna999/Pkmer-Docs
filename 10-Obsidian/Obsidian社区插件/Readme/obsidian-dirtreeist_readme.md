# Obsidian Dirtreeist

从代码块中的 markdown 列表中渲染一个目录结构图。

## 注意

该插件依赖于 [dirtreeist](https://github.com/k4a-l/dirtreeist) 库的功能。请将功能请求发送到该库。

## 特点

### 基本

如果你编写的代码块如下所示，

````
```dirtree
- /components
	- App.tsx
	- App.css
- config.json
- /utils
	- converter.ts
	- parser.ts
```
````

它将被渲染为以下形式，

```
├─ /components
│　├─ App.tsx
│　└─ App.css
├─ config.json
└─ /utils
　　├─ converter.ts
　　└─ parser.ts
```

### 其他

#### 顺序列表

```
- a
  - b
  - c
- d

- 1
  - 2
    - 3
      - 4
```

```
├─ a
│　├─ b
│　└─ c
├─ d
└─ 1
　　└─ 2
　　　　└─ 3
　　　　　　└─ 4
```

#### 另一个元素插入其中

```
- a
  - b
  - c
- d

一些文本

- 1
  - 2
    - 3
      - 4
```

```
├─ a
│　├─ b
│　└─ c
└─ d

└─ 1
　　└─ 2
　　　　└─ 3
　　　　　　└─ 4
```
