# 有序列表样式

这是一个适用于 [Obsidian](https://obsidian.md) 的插件。

在阅读模式下查看有序列表时，它们将根据缩进级别自动切换样式。默认设置了以下样式：

- 十进制
- 小写字母
- 小写罗马数字
- 大写字母
- 大写罗马数字
- 带前导零的十进制
- 实心圆点
- 空心圆
- 实心方块

请注意，这不会影响编辑模式，只会影响阅读模式。
