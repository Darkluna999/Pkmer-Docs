# Obsidian Markdown 链接空格编码器

一个简单的 Obsidian 插件，将选定文本中的空格编码为%20。对于 Vault 中的其他笔记的 Markdown 样式链接非常有用。

## 使用方法

1. 在 markdown 视图中，高亮显示链接的 markdown。
2. 打开命令面板（`ctrl+p`）。
3. 搜索并选择 `Markdown Link Space Encoder: Encode spaces to %20`。
