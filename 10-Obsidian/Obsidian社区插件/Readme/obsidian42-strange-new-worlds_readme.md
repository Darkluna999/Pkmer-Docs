奇异的网络思维新世界

该插件帮助您查看您的保险库中不同部分之间的连接。

基本思想是我们希望看到链接、块引用和嵌入与保险库中的其他文件有关联。问题是您必须搜索、打开反向链接等等才能了解情况。但是在我们的保险库中有许多奇异的网络思维新世界等待发现。该插件试图重新展现这些连接，并在此过程中不会过于侵入（或不会过于侵入）。
