# 标记和选择

这是一个 Obsidian 插件，提供了灵活的命令来选择文本区域。

用法

- 设置标记（Ctrl-@）
- 从标记处选择（Ctrl-#）

通过与其他光标导航方法结合使用，该插件可以非常强大。例如，要选择一个标题的内容，以下是步骤：

- 设置标记
- 通过 "Quick Switcher++: Open in Symbol Mode" 跳转到下一个标题
- 从标记处选择

这个插件在 Obsidian 移动端特别有用，因为长按 + 滚动选择长文本非常慢，而且选择很容易消失（可能是手机上的一个 bug）。
