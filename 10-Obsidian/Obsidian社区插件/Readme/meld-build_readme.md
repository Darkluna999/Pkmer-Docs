# Meld Build - 一个 Obsidian 插件

编写并执行（沙盒化的）JavaScript 代码来渲染模板、查询 DataView 并创建动态笔记。

基本上，将一个笔记转化为一个小而简单的可运行的东西。

- 安装并启用插件
- 将下面的 Markdown 粘贴到一个新的笔记中。
- 如果你在阅读或实时预览模式下，请点击“运行”按钮。如果你在源代码模式下，请从命令面板中选择“Meld Build: Run”。
