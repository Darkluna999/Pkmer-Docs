# Obsidian 自动隐藏插件

该插件使您能够更轻松地折叠（和展开）侧边栏。

## 演示

![演示](https://user-images.githubusercontent.com/97661658/184786896-358e253a-d024-4d31-a33b-f200ce69e53a.gif)

## 特点

- 单击笔记内容可折叠侧边栏。
- 如果您打开选项，可以通过以下方式展开侧边栏：
    - 单击左/右侧边栏的空白区域。
    - 单击笔记标题。
