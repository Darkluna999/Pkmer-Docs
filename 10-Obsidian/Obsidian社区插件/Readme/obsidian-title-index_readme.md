# 如何使用

按下 `ctrl + p`，然后输入 `add-markdown-index`。

# 功能

markdown_index 是一个可以为你的 markdown 标题添加序号的扩展。

假设原始文本如下：

```

# a
foolbar

## a-b
foolbar

愚蠢的酒吧
愚蠢的酒吧

### a-c-d
foolbar

```

    这是代码块

```

foolbar

    这也是一个代码块

# f
foolbar
```

安装 markdown_index，运行 > markdown add index，可以自动添加序号，如下所示：

```

1. a
foolbar

1.1. a-b
foolbar

1.2. a-c
foolbar
foolbar

### 1.2.1. a-c-d
foolbar

```

    这是代码块

```

1.3. a-e
foolbar

    这也是一个代码块

# 2. f
愚蠢的酒吧
```
