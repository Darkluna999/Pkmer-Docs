# Obsidian 的全局热键

这是一个 [Obsidian](https://obsidian.md) 插件，为系统范围内的全局热键提供支持。

## 特点

- 可以映射任何 Obsidian 命令
- 每个命令都会将 Obsidian 置于前台
- 添加了额外的命令：
  - 将 Obsidian 置于前台
  - 显示/隐藏 Obsidian

## 设置

在“全局热键”设置中，您可以输入一个系统范围的键盘组合，以执行指定的命令。键盘组合是根据 [Electron的加速器](https://www.electronjs.org/docs/api/accelerator) 来指定的。
