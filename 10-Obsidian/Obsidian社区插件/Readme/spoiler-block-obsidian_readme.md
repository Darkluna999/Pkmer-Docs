如何安装

### 从 Obsidian 内部

从 Obsidian v0.15.0+ 开始，您可以通过以下步骤在 Obsidian 内部激活此插件：

- 打开设置 > 第三方插件
- 确保安全模式处于关闭状态
- 点击浏览社区插件
- 搜索“Spoiler Block”
- 点击安装
- 安装完成后，关闭社区插件窗口并激活新安装的插件
