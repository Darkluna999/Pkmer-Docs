# Slackify Note

该插件添加了以下命令：

* `Slackify Note: 将笔记复制为 Slack Markdown` - 将笔记转换为符合 Slack 标准的 Markdown 格式。
* `Slackify Note: 将选定内容复制为 Slack Markdown` - 将当前选定内容转换为符合 Slack 标准的 Markdown 格式。

[安装此插件](https://obsidian.md/plugins?id=slackify-note)
